# Huzaifa Malik | AI & ML Engineer Portfolio

A premium, immersive space-themed portfolio built with **Next.js 14**, **Three.js**, and **Framer Motion**. Featuring a neural AI assistant, dynamic 3D visuals, and a deep-space aesthetic tailored for an AI Engineer.

## 🚀 Key Features

- **🌌 Immersive 3D Space Background**: Interactive starfield and black hole visuals powered by Three.js and `@react-three/fiber`.
- **🤖 Neural AI Assistant**: A custom-trained AI assistant with a WhatsApp-style "unroll" interface to answer questions about projects and skills.
- **📱 Responsive Design**: Fully optimized for all screen sizes with a mobile-first approach.
- **✨ Smooth Animations**: Complex entry and scroll animations using Framer Motion.
- **🔒 Secure Contact Form**: Integrated with Web3Forms for seamless communication.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **3D Visuals**: [Three.js](https://threejs.org/) & [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/)
- **Icons**: [Heroicons](https://heroicons.com/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Deployment**: [Vercel](https://vercel.com/)

## 📂 Project Structure

```bash
space-portfolio/
  |- app/           # Next.js App Router (Layouts & Pages)
  |- components/    # Reusable UI components
    |-- main/       # Major section components (Hero, Skills, Chatbot)
    |-- sub/        # Smaller sub-components and providers
  |- constants/     # Skills, Projects, and Social data
  |- public/        # Static assets (Images, Videos, SVGs)
  |- lib/           # Animation helpers and utility functions
```

## ⚙️ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Techsynx/exhuzaifa.git
   ```

2. **Install dependencies:**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Set up Environment Variables:**
   Create a `.env.local` file and add your Groq API Key for the AI Assistant:
   ```env
   NEXT_PUBLIC_GROQ_API_KEY=your_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
